Programul rezolvare oferă câte un scor fiecărui element din lista de soluții posibile 
în funcție de cât de des apare fiecare literă pe fiecare poziție și alege elementul cu scorul
maxim să-l dea ca input programului wordle, și în baza feedback-ului din programul wordle,
rezolvare elemină cuvintele care nu pot fi soluții. 

Programele wordle.py și rezolvare.py numără încercările necesare pentru fiecare cuvânt.
Programele wordle_pentru_cuvant_random.py si rezolvare_pentru_cuvant_random.py numără 
încercările necesare pentru un cuvănt random.

Comunicarea între programe se face prin intermediul fișierelor IO1.txt și IO2.txt.

În medie a fost nevoie de 6.13069669984285 soluții pentru ghicirea tuturor cuvintelor.
